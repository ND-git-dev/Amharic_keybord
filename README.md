# Amharic Keyboard

A custom Amharic keyboard application built using **Android Studio 3.5**.  
This project provides a simple way to type in Amharic on Android devices.

## 📱 Features
- Supports Amharic characters and input
- Built with Java and Android SDK
- Compatible with Android Studio 3.5
- Gradle-based build system

## 🛠️ Requirements
- Android Studio 3.5 or higher
- JDK 8 or higher
- Android SDK (API level 21+ recommended)
